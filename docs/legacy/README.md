# Legacy Materials

本目录收纳 AIPD2 演进过程中的历史材料。这里的内容保留为参考，不再作为当前实现和当前 Agent 路由入口。

## 目录

- `v1/`：AIPD v1 旧版框架资料、旧 skill 和旧 references。
- `v2-todo/`：AIPD2 早期待讨论事项与已完成事项归档。当前有效事项应以 `_adoc/inbox.md`、`_adoc/case/` 和 `_adoc/map.md` 为准。

## 使用规则

- 当前开发和 Agent 路由不要直接依赖本目录。
- 如果 legacy 中的信息重新变得有价值，先转入 `_adoc/inbox.md` 或创建新 case，再判断是否 weave 到 L1-L5。
