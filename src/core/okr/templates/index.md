# OKR 索引

OKR 用来记录项目阶段目标，帮助判断 Case 是否真的推进了项目方向。

## 当前 OKR

| OKR | 周期 | 状态 | 入口 |
|-----|------|------|------|
| 暂无 | - | 未设置 | - |

## 历史 OKR

| OKR | 周期 | 结果 |
|-----|------|------|
| 暂无 | - | - |

## 使用规则

- OKR 不必和 Case 一一对应。
- 一个 KR 可以由多个 Case 推进。
- 一个 Case 也可以同时推进多个 KR。
- 回顾时关注：这段时间完成的 Case 是否让项目更接近当前 KR。

## 推荐结构

```text
_adoc/okr/
├── index.md
└── O1-目标名/
    ├── index.md
    └── YYYY-MM-DD.md
```
