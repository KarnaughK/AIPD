# OKR 索引

OKR 用来记录项目阶段目标，帮助判断 Case 是否真的推进了项目方向。AIPD 里提到 OKR 时，默认就是飞书 OKR。

## 使用规则

- OKR 不必和 Case 一一对应。
- 一个 KR 可以由多个 Case 推进。
- 一个 Case 也可以同时推进多个 KR。
- 回顾时关注：这段时间完成的 Case 是否让项目更接近当前 KR。
- Agent 只需要检查 `lark-cli` 是否可用；不可用时提示用户安装。
